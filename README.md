## Boilerplate for Rails and PostgresSQL and Docker compose
Ruby 2.6.5  
Rails 6.0.3  
PostfresSQL 9.4.5  
Docker Compose 2.1  
  
  
`clone this repo`  
`chmod +x config/container.sh`  
`vim .env` to change credentials  
`vim dockercompose.yml` to change db file storage location on your machine
`docker-compose up`
